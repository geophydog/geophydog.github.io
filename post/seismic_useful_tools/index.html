⬇️ Travel time, ray parameter and ray path calculation

  • 0️⃣ 1️⃣ TauP Travel time, incident, takeoff, ray parameter and ray path calculation and show.

  • 0️⃣ 2️⃣ ObsPy TauP TauP implemented by ObsPy.

  • 0️⃣ 3️⃣ Cake Cake is a tool which can be used to solve classical seismic ray theory problems for layered earth models (layer cake models).

  • 0️⃣ 4️⃣ FM3D 3D FAST MARCHING CODE.

  • 0️⃣ 5️⃣ pySeismicFMM Calculating 3-D travel times by FAST MARCHING.

  • 0️⃣ 6️⃣ ANISOtime Calculating travel times in homogeneous layers, transversely isotropic (TI), spherically symmetric medium.

⬇️ Visualization

  • 0️⃣ 1️⃣ GMT
    Almost the most usual mapping software in geophysical field.

  • 0️⃣ 2️⃣ matplotlib
    Not only in seismology, it's useful to plot beautiful figures.

  • 0️⃣ 3️⃣ SAC
    Plot simple figures and seismic data processing.

  • 0️⃣ 4️⃣ PSSAC
    Show seismic waveforms calling GMT coomand.

  • 0️⃣ 5️⃣ POV-Ray
    Apply colors to drawing and it's beautiful.

  • 0️⃣ 6️⃣ ECharts
    Awesome figures, tables building created by BAIDU Inc and implemented by JavaScript.

  • 0️⃣ 7️⃣ PyECharts
    A python library implementing ECHarts.

  • 0️⃣ 8️⃣ObsPy
    Spectragram, beachball or seismic data processing.

  • 0️⃣ 9️⃣ Snuffler
    View seismic waveforms.

  • 1️⃣ 0️⃣ Desmos
    Demo of graphs of mathematical functions.

  • 1️⃣ 1️⃣ GeoGebra Geometry Algebra, a amazing tool.

  • 1️⃣ 2️⃣ Mathematica

    Data processing, visualization, numerical and symbolic computation.

  • 1️⃣ 3️⃣ Obtaining colors

    You can get these colors on line when you upload your picture

  • 1️⃣ 4️⃣ Choosing colors

    There are many colors and you can choose them if necessary.

  • 1️⃣ 5️⃣ M_Map

    A Matlab packge for geographical mapping tool.

⬇️ Synthetic Seismograms

  • 0️⃣ 1️⃣ Fomosto Computing synthetic and store seismograms.

  • 0️⃣ 2️⃣ fk Computing synthetic seismograms in horizontal layered medium.

  • 0️⃣ 3️⃣ DSM Synthesizing seismograms in homogeneous layers, transversely isotropic (TI), spherically symmetric medium with Direct Solution Method (DSM).

  • 0️⃣ 4️⃣ yaseis Computing synthetic seismograms inhomogeneous layers, transversely isotropic (TI), spherically symmetric medium.

  • 0️⃣ 5️⃣ Mineos Computing synthetic seismograms in a spherically symmetric non-rotating Earth by summing normal modes.

  • 0️⃣ 6️⃣ SOFI2D Computing 2-D PSV wavefield in viscoelastic medium with finite difference method.

  • 0️⃣ 7️⃣ SOFI2D-sh Computing 2-D SH wavefield in viscoelastic medium with finite difference method.

  • 0️⃣ 8️⃣ SOFI3D ` Computing 3-D wave field in viscoelastic medium with finite difference method.

  • 0️⃣ 9️⃣ sw4 3-D seismic modeling with finite difference method.

  • 1️⃣ 0️⃣ OpenSWPC 2/3-D seismic wave propagation modeling with finite difference method.

  • 1️⃣ 1️⃣ AxiSEM AxiSEM is a parallel spectral-element method to solve 3D wave propagation in a sphere with axisymmetric or spherically symmetric visco-elastic, acoustic, anisotropic structures.

  • 1️⃣ 2️⃣ NEXD NEXD: A SOFTWARE PACKAGE FOR HIGH ORDER SIMULATION OF SEISMIC WAVES USING THE NODAL DISCONTINUOUS GALERKIN METHOD.

  • 1️⃣ 3️⃣ PRINCETON GEODYNAMICS SOFTWARES

  • 0️⃣ 1️⃣ gCAP Inversion of seismic source parameters with generalized Cut And Paste (gCAP) method.

  • 0️⃣ 2️⃣ pydmt Time Domain Moment Tensor Inversion,

  • 0️⃣ 3️⃣ HASH First motion polarity.

  • 0️⃣ 4️⃣ focmec Determine and display focal mechanism.

  • 0️⃣ 5️⃣ WPhase W Phase source inversion. Fast and reliable moment tensor estimation.
